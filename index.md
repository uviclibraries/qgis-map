# Introduction to Vector with *QGIS*

- Pre-workshop activities: _ min 
- Introductory presentation: 30 min
- Hands-on activities: 60-90 min

## Why QGIS? 

[QGIS]([https://qgis.org/download/){:target="_blank"}

QGIS is a free, open source geographic information system (GIS) software that allows users to analyze and edit spatial information and create and export maps.

## Learning objectives
1. Distinguish between GIS and other web map interfaces
2. Identify and navigate key QGIS interface elements (Layers panel, Menu bar, Map view)
3. Define the basics of vector data
4. Explore data layers using tools such as identify feature

## Learning Outcomes
Using *QGIS*, participants will:

1. Load and display vector data from the Capital Regional District (CRD neighbourhoods (polygon), bus routes (lines), location coordinate (points))
2. Import locations.csv data and add to it
3. Style above
4. Export map
